# 🎨 Background Color Changer

Changes the background color to a random HEX color on button click.

## 🔧 Features
- Random color generation
- Display HEX color code

## 💡 Technologies
- HTML
- CSS
- JavaScript

## 🖥️ How to Run
Open `index.html` in your browser.

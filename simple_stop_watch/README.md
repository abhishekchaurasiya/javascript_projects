# ⏱️ Simple Stop Watch

A stopwatch with start, stop, and reset functionality.

## 🔧 Features
- Start/Stop timer
- Reset to 00:00
- Millisecond precision

## 💡 Technologies
- HTML
- CSS
- JavaScript

## 🖥️ How to Run
Open `index.html` in your browser.

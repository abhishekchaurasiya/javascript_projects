# 🚀 Auto Text Generator

Automatically types and re-types a given text in a typing animation loop.

## 🔧 Features
- Typing effect animation
- Adjustable typing speed
- Loops infinitely

## 💡 Technologies
- HTML
- CSS
- JavaScript

## 🖥️ How to Run
Open `index.html` in your browser.

# 📜 Page Scroller Indicator

Visual scroll indicator bar that shows how much of the page has been read.

## 🔧 Features
- Scroll tracking
- Fixed progress bar

## 💡 Technologies
- HTML
- CSS
- JavaScript

## 🖥️ How to Run
Open `index.html` in your browser.
